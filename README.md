# Enterprise Order Routing System

A modern, full-stack Enterprise Resource Planning (ERP) sub-module designed to streamline the order lifecycle, automate store distribution routing, and manage inventory allocation. Built with a performance-first mindset utilizing an asynchronous .NET Web API and a highly responsive React single-page application.


## 🛠️ Tech Stack

### Backend
* **Runtime & Language:** .NET 10.0 / C#
* **Framework:** ASP.NET Core Web API
* **ORM:** Entity Framework Core (EF Core)
* **Database Provider:** PostgreSQL
* **Logging:** Serilog with Request Logging middleware

### Frontend
* **Library:** React (TypeScript)
* **Build Tool:** Vite
* **Data Fetching:** TanStack Query
* **Styling:** Modern Responsive CSS / TailwindCSS

### DevOps & Infrastructure
* **Containerization:** Docker & Docker Compose
* **Database Engine:** PostgreSQL 16 (Alpine Distribution)
* **Web Server / Reverse Proxy:** Nginx (Production Routing)
